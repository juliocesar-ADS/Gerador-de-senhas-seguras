🔐 Gerador de Senhas Seguras (Python)

Um projeto simples em Python para gerar senhas seguras diretamente pelo terminal. Ideal para iniciantes que querem praticar lógica de programação, loops e manipulação de strings.

🚀 Funcionalidades
✅ Gerar uma ou várias senhas seguras
✅ Definir o tamanho da senha
✅ Armazenar senhas geradas durante a execução
✅ Visualizar todas as senhas criadas
✅ Interface simples no terminal
